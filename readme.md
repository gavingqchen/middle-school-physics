# 初中物理

## 说明
虽然毕业这么多年，四大力学等书籍依旧在书架之上，然而复杂的推导却早已忘却，仅留下一丝概念残留脑海以作纪念。
趁着辅导的机会，慢慢整理整理中学的物理知识，也算是对物理的一种情怀吧。

## 已包含内容
* U1 测量
* U2 声
